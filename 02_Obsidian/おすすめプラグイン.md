# Obsidian おすすめプラグイン

ジャンル別に、よく使われる・評価の高いコミュニティプラグインをまとめています。まずは2〜3個から試し、必要に応じて増やすのがおすすめです。

---

## 定番の「鉄板」3つ

多くのユーザーが「必須」として挙げる組み合わせです。

| プラグイン | 用途 |
|------------|------|
| **Templater** | テンプレートの高度化。日付・時刻の自動挿入、前日・翌日リンク、JavaScript スニペット、ユーザー入力など。デイリーノートや定型ノートの雛形に最適。 |
| **Dataview** | ノートをデータのように扱う。タグ・frontmatter・日付でクエリし、表・リスト・カレンダー表示を**動的**に生成。ダッシュボード・索引・読書リスト向き。 |
| **Calendar** | サイドバーにカレンダーを表示。日付をクリックしてデイリーノートを開く。デイリーノート運用と相性が良い。 |

## ジャンル別おすすめ

### タスク管理

| プラグイン | 用途 |
|------------|------|
| **Tasks** | チェックボックス（`- [ ]`）をタスクとして集約。期限・繰り返し・フィルター表示。Dataview と組み合わせて一覧化できる。 |
| **Kanban** | Trello 風のカンバンボード。ノートや見出しをカードとしてドラッグ＆ドロップで管理。 |
| **Checklist** | チェックリストの表示・操作を強化。 |

### ノート整理・検索

| プラグイン | 用途 |
|------------|------|
| **Quick Switcher++** | クイックスイッチャーを強化。曖昧検索・最近使ったノート・高速ジャンプ。 |
| **Outliner** | アウトライン（見出し・リスト）の階層操作を強化。折りたたみ・移動・インデントがしやすい。 |
| **Tag Wrangler** | タグの一括リネーム・マージ・削除。タグ整理に便利。 |

### デイリーノート・周期ノート

| プラグイン | 用途 |
|------------|------|
| **Periodic Notes** | 日・週・月・年単位のノートをテンプレート付きで作成。Calendar と連携するとスムーズ。 |
| **Review** | ノートを「○日後にもう一度見る」ようにリマインダー。間隔反復の復習向き。 |

### 図・ビジュアル

| プラグイン | 用途 |
|------------|------|
| **Excalidraw** | 手書き風の図・フローチャート・マインドマップをノート内に埋め込み。 |
| **Mind Map** | 見出し構造をマインドマップとして表示。 |
| **Graph Analysis** | グラフビューを拡張。ノート間のつながりを分析。 |

### 執筆・編集効率

| プラグイン | 用途 |
|------------|------|
| **Advanced Tables** | Markdown の表をキーボードで編集。セル移動・行追加が楽になる。 |
| **QuickAdd** | テンプレート挿入・新規ノート作成・マクロをショートカットやコマンドで実行。キャプチャワークフロー向き。 |
| **Auto Note Mover** | 条件（タグ・フォルダなど）に応じてノートを自動でフォルダ移動。 |

### 日本語・表示

| プラグイン | 用途 |
|------------|------|
| **Hard Breaks** | 日本語の改行挙動を改善（1改行で1行として表示するなど）。 |
| **Jisage Japanese Indentation** | 日本語の段落頭（字下げ）を自動調整。 |

### その他ユーティリティ

| プラグイン | 用途 |
|------------|------|
| **Homepage** | 起動時に指定したノートをホーム画面として表示。 |
| **LazyPlugin** | 起動時にプラグインの読み込みを遅延させ、起動を軽くする。 |

## インストール方法（コミュニティプラグイン）

1. **設定**（歯車）→ **コミュニティプラグイン** を開く。
2. **「Turn on community plugins」** でセーフモードをオフにする（初回のみ）。
3. **閲覧** でプラグイン名を検索 → **インストール** → **有効化**。
4. 必要に応じてプラグインの設定画面でフォルダ・テンプレート・ショートカットを指定する。

※ 更新は同じ画面から手動で行います。

---

## 選ぶときのポイント

- **少なめから始める**: まずは Templater・Calendar・Tasks など 2〜3 個に絞ると慣れやすい。
- **公式ディレクトリを優先**: 更新頻度や評価・レビューを確認してから入れる。
- **データは Markdown のまま**: 多くの人気プラグインは .md を編集するだけなので、ロックインされにくい。
- **重要情報は控えめに**: プラグインはローカルで動くが、信頼できるものに留め、極秘情報は別管理も検討。

## このVaultでの利用例

- **00_Daily** … Calendar で日付クリック、Templater でテンプレート拡張。
- **03_Template** … Templater のテンプレートフォルダに `03_Template` を指定すると、`Daily_Template` をより柔軟にできる。
- **タスク一覧** … Tasks + Dataview で「今週の未完了タスク」などをノートに埋め込む。
