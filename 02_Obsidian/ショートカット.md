#tool/obsidian
Obsidian（Mac）の標準ショートカットと、よく使われる追加ショートカットを、Mac専用のキー表記だけで箇条書きします。

## ノート作成・ファイル操作（Mac）

- 新規ノート作成：Cmd + N
- ノートを保存（自動保存なので基本不要だが一応）：Cmd + S
- ノートを別タブで開く：Cmd + Enter（リンク上で）
- 現在のタブを閉じる：Cmd + W
- すべてのタブを閉じる：Cmd + Shift + W

## 検索・移動系

- クイックスイッチャー（ノート素早く移動）：Cmd + O
- コマンドパレット：Cmd + P
- ノート内検索：Cmd + F
- すべてのノートを検索：Cmd + Shift + F
- バックリンクペイン表示：Cmd + Shift + B

## 表示切り替え・画面操作

- 編集／プレビュー切り替え：Cmd + E
- サイドバー表示切り替え（左）：Cmd + Option + L（カスタム推奨）
- サイドバー表示切り替え（右）：Cmd + Option + R（カスタム推奨）
- 設定を開く：Cmd + ,（カンマ）

## タブ・ペイン操作

- タブ間移動（左／右）：Ctrl + Tab / Ctrl + Shift + Tab（ブラウザと同様）
- スタックタブの切り替え：Cmd + Shift + Option + S（カスタム例）
- ペインを分割して開く：Cmd + Option + Enter（リンク上で、カスタムにする人が多い）

## 文章作成・Markdown編集

- 太字：Cmd + B
- 斜体：Cmd + I
- リンク挿入（URLリンク）：Cmd + K
- 箇条書きトグル：Cmd + Shift + 8（Mac標準のリスト）
- 番号付きリスト：Cmd + Shift + 7
- チェックボックス（タスクリスト）：Cmd + L（または好みのキーへ割当推奨）
- 引用ブロック：Cmd + Option + Q
- コールアウト（info など）：Cmd + Shift + I（カスタム例）
- 行の上下移動：Option + Cmd + ↑ / ↓

## 見出し・構造化

- 見出しレベル変更：Cmd + Option + 1〜6（H1〜H6）
- 箇条書きインデント／アウトデント：Cmd + ] / Cmd + [

## リンク・ナレッジグラフ系

- ノートリンクを挿入：[[ とノート名入力（自動補完）
- 外部リンク挿入（Cmd + K）：選択テキストをリンク化
- ファイルURLコピー：Cmd + Shift + U（カスタム例。Obsidian URL コピーに割り当て）
- グラフビューを開く：Cmd + G（カスタム推奨）

## デイリーノート・テンプレート周り（よく使われるカスタム）

- 今日のデイリーノートを開く：Cmd + Option + D などに割当（「Daily notes」コマンド）
- テンプレート挿入：Option + T などに割当（「Insert template」コマンド）
- 日付入りノート（ジャーナル）用テンプレート呼び出し：Hyperキー＋J 等（macOSユーティリティでHyperキー設定）

## ショートカットの確認・追加方法（Mac）

- 現在使えるショートカットを一覧表示：Cmd + P → コマンドパレットから確認
- 設定 → Hotkeys でコマンドを検索し、好きなキーを割り当て可能（Macでは Cmd／Option／Ctrl を組み合わせる）

もし「デイリーノート中心」「Zettelkasten」「タスク管理重視」など、Obsidianの主な使い方を教えてもらえれば、その用途に最適化した「これだけ覚えればOKなMacショートカット」のセットも作ります。
