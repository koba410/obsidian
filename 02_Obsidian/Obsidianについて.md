# Obsidianについて

## 概要

**Obsidian**は、**マークダウン形式のノートをローカルで管理する**アプリケーションです。「思考を研ぎ澄ます」をコンセプトに、個人の知識管理（PKM）や「第二の脳」づくりに使われることが多いツールです。

- **キャッチコピー**: "Sharpen your thinking"（思考を研ぎ澄ます）
- **公式説明**: プライベートな思考のための、無料で柔軟なアプリ
- **対応OS**: Windows / macOS / Linux / iOS / Android

## 主な特徴

### 1. 双方向リンク（バックリンク）

- ノート同士を `[[ノート名]]`（ウィキリンク）でリンクできる
- リンク先から「このノートを参照しているノート」が自動で把握できる（バックリンク）
- 個人用Wikipediaのように、アイデア・人物・場所・本など何でもリンクでつなげられる

### 2. グラフビュー（Graph View）

- ノート間のつながりを**ネットワーク図**として可視化
- 思考のパターンや関連の強いノートを視覚的に発見できる

### 3. ローカルファースト・オープンな形式

- データは**すべてローカル**のマークダウンファイル（.md）
- **プライバシー**: 端末内に保存され、オフラインでも利用可能。Obsidian側が内容を読むことはない
- **データの所有**: フォーマットがオープンなので、他ツールへの移行や長期保存に向いている

### 4. 高いカスタマイズ性

- **プラグイン**: 数千本のコミュニティプラグイン（2025年時点で1,500本以上）
- **テーマ**: 見た目を自由に変更可能
- **オープンAPI**: 自分でプラグインやテーマを開発できる（[Developer docs](https://docs.obsidian.md/)）

### 5. キャンバス（Canvas）

- 無制限のキャンバス上で、ノート・画像・リンクを配置してブレストや図解ができるコア機能
- リサーチ・アイデア整理・図解に利用される

---

## コア機能の補足

| 項目 | 内容 |
|------|------|
| **リンク形式** | ウィキリンク `[[ノート名]]`（標準）、Markdown形式 `[表示名](ファイル名)` も利用可能。設定でウィキリンクを無効化可能 |
| **リンクの作り方** | `[[` と入力してファイル名を補完、またはテキスト選択後に `[[`、コマンドパレットから「内部リンクを追加」 |
| **ブロックリンク** | 見出しや特定ブロックへのリンク、`!` 付きでコンテンツの埋め込みも可能 |
| **保管単位** | **Vault（ボールト）**: 1つのフォルダ＝1つの知識ベース。その中に .md ファイルを配置 |

## 料金体系

| 種別 | 内容 |
|------|------|
| **基本アプリ** | **無料**（個人・商用とも。2025年2月から商用利用も無料化） |
| **Obsidian Sync** | 有料（月額目安 8〜10ドル程度）。端末間同期・エンドツーエンド暗号化・バージョン履歴（1年分）・チーム用コラボ |
| **Obsidian Publish** | 有料（月額目安 16ドル程度）。ノートをWebサイト・Wiki・ドキュメントとして公開。テーマ・カスタムドメイン・パスワード保護など |

## 活用シーン（公式・コミュニティより）

- 個人メモ・日記（ジャーナリング）
- ナレッジベース・個人Wiki
- プロジェクト管理・タスク管理（プラグインと組み合わせ）
- リサーチ・読書メモ・Zettelkasten 的なノート術
- 公開用：デジタルガーデン・ドキュメントサイト（Publish 利用時）

---

## 他ツールとの比較（イメージ）

- **Notion**: チーム共有・データベース・リアルタイム共同編集に強い。Obsidianは**個人の知識資産**と**ローカル・オープン形式**を重視する用途に向く
- **Evernote / OneNote**: クラウド中心のノート。Obsidianはローカル＋リンク＋グラフで「つながり」を扱う設計
- **Scrivener / Ulysses**: 長文・執筆特化。Obsidianは短文ノートのネットワークと知識管理に適する

## まとめ

Obsidianは、**マークダウンのローカルノート**を**双方向リンクとグラフ**でつなぎ、プラグインで拡張できる**個人向け知識管理アプリ**です。データは自分で持ち、オフラインでも使え、長期的な資産として運用しやすい点が特徴です。
